# Project Instructions

## Commands

```bash
pnpm dev        # dev server
pnpm build      # production build + type check
pnpm lint       # ESLint
```

## Workflow

- `pnpm build` type-checks too — run after a series of changes
- Prefer fixing the root cause over adding workarounds
- When unsure, use plan mode (`Shift+Tab`) before coding

## Don'ts

- Don't modify `next-env.d.ts` (auto-generated by Next.js)
